# Project Overview

## Vision
Free, accessible Lithuanian language learning for everyone.
Smooth, engaging UX. Offline-friendly. Scalable content.

## Target Users
- International students in Lithuania
- Immigrants and workers
- Travelers and heritage learners
- Anyone wanting to learn Lithuanian from scratch

## Value Proposition
- **Free**: No paywalls for core learning content
- **Offline-first**: Learn without internet after initial download
- **Structured**: CEFR-aligned progression (A1 now, A2 later)
- **Engaging**: Gamified lessons, spaced repetition, achievement system

## Platforms
| Platform | Status | Notes |
|----------|--------|-------|
| Android | Primary | MVP target |
| iOS | Future | After Android launch |
| Web | Future | For accessibility |

## Non-Goals
- We are NOT an official CEFR certification body
- We do NOT replace classroom instruction
- We do NOT provide real-time human tutoring
- We do NOT monetize user data

## Success Metrics (Future)
- Daily active users
- Lesson completion rates
- User retention (D1, D7, D30)
- Content coverage (lessons per unit)
