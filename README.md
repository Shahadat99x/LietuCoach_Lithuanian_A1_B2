# LietuCoach - Lithuanian Language Learning App

A free, offline-friendly Lithuanian language learning app for Android.
Target users: international students, immigrants, travelers. Starting with A1 level.

## Tech Stack
- **Mobile**: Flutter (Dart)
- **Content**: Versioned JSON packs delivered via Play Asset Delivery (PAD)
- **Auth/Sync**: Supabase (Google OAuth + optional email)
- **Audio**: Pre-generated TTS files (no on-demand TTS)

## Quick Start
```bash
flutter pub get
flutter run
flutter analyze
flutter test
```
